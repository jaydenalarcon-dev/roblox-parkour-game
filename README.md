# roblox-parkour-game
Published Roblox parkour game developed in Luau with custom (soon to be procedully generated) levels, movement mechanics, and gameplay systems.

How to play:
1. Go to https://www.roblox.com/ and create an account! (if you dont have one already)
2. Go to https://www.roblox.com/games/104335870242946/PAF-ASK (roblox game link) and press the blue play button
3. Download roblox and you should be able to play!

Controls
PC:
WASD - Move character
Shift - Run
Space - Jump

Mobile:
Walk Joystick - Move character
Up button - jump
Toggle Run - (To be programmed
